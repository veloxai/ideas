## ğŸ¯ Market Opportunity

### Target Audience

**Primary Users**:

1. **Newly Diagnosed Alpha-1 Patients** (1,000+/year)
   - Just learned diagnosis, confused about family risk
   - Want genetic counseling to understand implications
   - Need help coordinating family screening

2. **Family Members of Alpha-1 Patients** (10,000+ at risk)
   - Siblings: 50% chance of carrier or affected
   - Children: 100% carrier if patient is PI*ZZ (50% carrier if PI*MZ)
   - Cousins, aunts/uncles: Lower risk but still relevant

3. **Carriers (PI*MZ, PI*MS)** (3 million in US)
   - Most don't know they're carriers
   - Want to know: "Should I tell my kids? My siblings?"
   - Reproductive planning (if partner is also carrier, 25% chance of PI\*ZZ child)

4. **Prospective Parents** (couples considering children)
   - One or both partners are carriers
   - Want preconception counseling
   - Consider IVF + PGD (preimplantation genetic diagnosis) to avoid PI\*ZZ

**Secondary Beneficiaries**:

- Genetic counselors (new revenue stream, flexible work)
- Alpha-1 researchers (access to genotype-phenotype database)
- Pharmaceutical companies (identify undiagnosed patients for clinical trials)

### Market Size

**Alpha-1 Patients in US**: 100,000-200,000
**Diagnosed**: 10,000-15,000 (~5-7.5%)
**New diagnoses per year**: 1,000-1,500

**Family Members at Risk**:

- Each diagnosed patient has ~20 first-degree relatives (siblings, children)
- Of those, ~10-15 should get tested (based on inheritance patterns)
- **Total at-risk family members**: 100,000+ (10 per patient Ã— 10,000 diagnosed)

**Current Genetic Counseling Access**:

- Patients who receive counseling: ~5% (500 of 10,000)
- Family members who get tested: ~20% (20,000 of 100,000)

**Opportunity**: 95% of patients lack genetic counseling, 80% of families untested

### Competitive Landscape

**Existing Solutions**:

1. **Hospital Genetic Counseling Services**:
   - Limited availability (4,500 genetic counselors in US, mostly cancer-focused)
   - Long wait times (2-6 months for appointment)
   - Expensive ($300-$500, often not covered by insurance)
   - **Limitation**: Not Alpha-1 specific, generic advice

2. **Alpha-1 Foundation Family Screening Program**:
   - Free screening events (in-person, pre-COVID)
   - Mail-in DNA testing (still offered)
   - **Limitation**: No genetic counseling component, slow turnaround (6-8 weeks)

3. **23andMe, AncestryDNA** (Direct-to-Consumer):
   - Detect _some_ Alpha-1 variants (not comprehensive)
   - No counseling, confusing results
   - **Limitation**: Miss rare variants, false reassurance

4. **InformedDNA, GeneCounsel** (Telehealth genetic counseling):
   - Virtual appointments ($250-$400)
   - Not disease-specific
   - **Limitation**: Not integrated into Alpha-1 community

- âœ… Affordable ($150-$200, vs. $300-$500 elsewhere)
- âœ… Fast (appointments within 1-2 weeks, not months)

**Market Gap**: No one offers comprehensive genetic counseling + family screening coordination specifically for Alpha-1.

async function orderQuestTest(patientInfo, insuranceInfo) {
  const orderPayload = {
    patient: {
      firstName: patientInfo.firstName,
      lastName: patientInfo.lastName,
      dateOfBirth: patientInfo.dob,
      sex: patientInfo.sex,
      address: patientInfo.address,
      phone: patientInfo.phone,
      email: patientInfo.email
    },
    insurance: {
      payorName: insuranceInfo.payorName,
      policyNumber: insuranceInfo.policyNumber,
      groupNumber: insuranceInfo.groupNumber
    },
    tests: [
      {
        testCode: "35571N", // Quest test code for Alpha-1 genotype
        icd10Codes: ["E88.01"], // Alpha-1 antitrypsin deficiency
        clinicalInfo: "Family history of Alpha-1 antitrypsin deficiency"
      }
    ],
    orderingProvider: {
      npi: process.env.MEF_PROVIDER_NPI, // MEF medical director
      name: "Dr. John Smith"
    },
    specimenCollection: {
      method: "mail_in_kit", // or "psc" (patient service center)
      kitShippingAddress: patientInfo.address
    }
  };

  const response = await fetch('https://api.questdiagnostics.com/v1/orders', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${QUEST_API_KEY}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify(orderPayload)
  });

  const order = await response.json();

  // Store order ID in database
  await db.query('INSERT INTO dna_test_orders (user_id, lab, order_id, test_code, status) VALUES (?, ?, ?, ?, ?)',
    [patientInfo.userId, 'quest', order.orderId, "35571N", 'pending']);

  return order;
}

  // Extract genotype from Quest report
