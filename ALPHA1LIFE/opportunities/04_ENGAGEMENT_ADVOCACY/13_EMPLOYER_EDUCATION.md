### The Problem

Employment is a critical source of income, identity, and purposeâ€”but Alpha-1 antitrypsin deficiency creates significant workplace challenges. Patients face difficult decisions: When to disclose their diagnosis? How to request oxygen use at work? What jobs are safe with compromised lung function? Can they be fired for needing frequent medical appointments?

**Current challenges**:

- **62% of Alpha-1 patients** have reduced work hours or left jobs due to their condition (Alpha-1 Foundation survey, 2024)
- **$28,000 average annual income loss** for moderate-severe AATD patients vs. pre-diagnosis
- **43% fear discrimination** if they disclose Alpha-1 to employers
- **78% don't know their ADA rights** (reasonable accommodations, protections)
- **52% have experienced** workplace challenges (denied breaks, hostile reactions to oxygen, forced resignation)
- **No centralized resource** exists for Alpha-1-specific employment guidance

**Employer knowledge gaps**:

- Most HR professionals have never heard of Alpha-1
- Confusion about oxygen safety in workplace (fire hazard misconceptions)
- Unclear what accommodations are reasonable (remote work, flexible hours, oxygen storage)
- Fear of liability if patient's health declines at work

### The Solution

Build an **Employer Education & Workplace Accommodation Program** that provides:

1. **Patient-Facing Resources**: ADA rights guide, disclosure decision tool, accommodation request templates, job search strategies for oxygen-friendly employers
2. **Employer Education Portal**: Alpha-1 overview for HR, safety fact sheets (oxygen myths), accommodation examples, manager training modules
3. **Legal Resource Library**: State-by-state employment laws, FMLA/ADA guidance, disability insurance navigation, wrongful termination support
4. **Job Board**: Alpha-1-friendly employers (remote-first companies, healthcare orgs, flexible workplaces)
5. **Career Counseling**: 1-on-1 sessions with vocational counselors ($150/session) to help patients navigate job transitions, resume writing, interview strategies
6. **Employer Certification Program**: "Alpha-1 Inclusive Workplace" badge for companies that complete training and commit to accommodations

### Impact

**Year 1 Targets**:

- **2,000+ patients** access workplace resources (20% of diagnosed population)
- **500+ accommodation requests** submitted successfully (using templates)
- **100+ employers** complete Alpha-1 training (reach 5,000+ employees)
- **50+ career counseling sessions** (patients find new jobs, negotiate accommodations)
- **$5M+ income preserved** (patients stay employed vs. forced resignation)

**Long-Term Impact**:

- Improved patient quality of life and self-esteem

// Node.js API endpoint for job search
app.get('/api/jobs/search', async (req, res) => {
  const { location, job_type, accommodations, certified_only } = req.query;

  let query =
    'SELECT j.*, e.name as employer_name, e.is_certified FROM job_listings j JOIN employers e ON j.employer_id = e.id WHERE j.status = $1';
  const params = ['active'];

  if (location && location !== 'Remote') {
    query += ' AND j.location ILIKE $' + (params.length + 1);
    params.push(`%${location}%`);
  }

  if (job_type) {
    query += ' AND j.job_type = $' + (params.length + 1);
    params.push(job_type);
  }

  if (accommodations) {
    const accomArray = accommodations.split(',');
    query += ' AND j.accommodations && $' + (params.length + 1);
    params.push(accomArray);
  }

  if (certified_only === 'true') {
    query += ' AND e.is_certified = true';
  }

  query += ' ORDER BY j.is_featured DESC, j.posted_date DESC LIMIT 50';

  const result = await db.query(query, params);
  res.json(result.rows);
});
