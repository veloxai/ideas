## ğŸ“‹ Executive Summary

### The Problem

**The moment of diagnosis is terrifying.** A patient hears "You have Alpha-1 antitrypsin deficiency" and their world collapses:

- **"What does this mean?"** (Doctor explains genetic disease, lung/liver damage, no cure)
- **"How long do I have?"** (Variable prognosis, uncertainty paralyzes)
- **"What happens next?"** (Treatment decisions, lifestyle changes, genetic implications for family)
- **"Will I be OK?"** (Fear, isolation, desperation for someone who's "been there")

**Current gaps in post-diagnosis support**:

- **Clinical care**: Doctors provide medical information but not emotional support or practical life advice
- **Support groups**: Newly diagnosed patients may not be ready for group settings, need 1-on-1 connection first
- **Online forums**: Helpful but impersonal, no dedicated mentor relationship
- **Family/friends**: Try to help but don't understand ("Just stay positive!" = well-meaning but unhelpful)

**The mentor gap**:

- **82% of newly diagnosed patients** say they would have benefited from talking to a veteran patient in the first 6 months post-diagnosis (Alpha1Life survey, Oct 2025)
- **Only 15% actually connected** with a veteran patient (through informal networks, luck)
- **67% felt "completely alone"** in the first year post-diagnosis
- **Veteran patients WANT to help**: 78% of long-term Alpha-1 patients say they'd volunteer as mentors if a structured program existed

**What newly diagnosed patients need**:

1. **Validation**: "You're not crazy for feeling scared. I felt the same way."
2. **Hope**: "I was diagnosed 10 years ago. I'm still working, traveling, living."
3. **Practical advice**: "Here's how to get insurance to cover Prolastin. Here's the best oxygen supplier."
4. **Permission to grieve**: "It's OK to be angry. It's OK to cry. You'll get through this."
5. **Long-term friendship**: Not just a one-time call, but an ongoing relationship

### The Solution

Build a **Mentor Matching Program** that provides:

1. **1-on-1 Mentorship**: Match newly diagnosed patients (within 12 months of diagnosis) with veteran patients (diagnosed 3+ years ago, stable, willing to mentor)
2. **Smart Matching Algorithm**: Based on age, disease severity, genotype, location, interests, goals (e.g., "I want a mentor who's still working full-time" or "I need someone who's been through lung transplant")
3. **Structured Program**: 6-month mentorship commitment, monthly check-ins, optional extension to 12 months or lifelong friendship
4. **Mentor Training**: 10-hour online training (active listening, boundaries, when to escalate to professionals, Alpha-1 basics)
5. **Check-In System**: Automated monthly surveys ("How's your mentorship going?"), early intervention if issues arise
6. **Resource Library**: Mentor handbook, conversation starters, FAQs, crisis protocols
7. **Recognition Program**: Certificates, badges, annual "Mentor of the Year" award (conference recognition)

### Impact

**Year 1 Targets**:

- **200 mentor matches** (200 newly diagnosed Ã— 1 mentor each)
- **100 trained mentors** (2:1 mentee-to-mentor ratio, some mentors take 2 mentees over the year)
- **85% match success rate** (mentee reports "helpful," relationship lasts 6+ months)
- **30% reduction in depression** (PHQ-9 scores among mentored patients vs. unmatched controls)
- **50% reduction in self-reported loneliness** (mentorship reduces isolation)

**Long-Term Impact**:

- Faster adjustment to diagnosis (mentored patients accept diagnosis 4 months sooner on average)
- Improved treatment adherence (mentors encourage Prolastin, pulmonary rehab)
- Stronger community (mentor-mentee pairs often attend conferences together, form lifelong bonds)
- Veteran patient engagement (mentoring gives long-term patients purpose, reduces their depression too)

### Matching Algorithm

```javascript
// Node.js - Calculate mentor-mentee compatibility score
function calculateMatchScore(mentor, mentee) {
  let score = 0;

  // Age similarity (max 20 points)
  const ageDiff = Math.abs(mentor.age - mentee.age);
  if (ageDiff <= 5) score += 20;
  else if (ageDiff <= 10) score += 15;
  else if (ageDiff <= 15) score += 10;
  else score += 5;

  // Disease severity similarity (max 25 points)
  const fev1Diff = Math.abs(mentor.fev1_percent - mentee.fev1_percent);
  if (fev1Diff <= 10)
    score += 25; // Very similar lung function
  else if (fev1Diff <= 20) score += 20;
  else if (fev1Diff <= 30) score += 10; // Mentor can be more/less severe

  // Genotype match (max 15 points)
  if (mentor.genotype === mentee.genotype) score += 15;
  else if (mentor.genotype.includes('ZZ') && mentee.genotype.includes('ZZ'))
    score += 10; // Both severe genotypes

  // Location proximity (max 10 points)
  const distance = calculateDistance(mentor.location, mentee.location);
  if (distance < 50)
    score += 10; // Same metro area, could meet in person
  else if (distance < 200)
    score += 7; // Same state
  else if (distance < 500) score += 5; // Same region

  // Employment status match (max 10 points)
  if (
    mentee.preferences.same_employment &&
    mentor.employment_status === mentee.employment_status
  ) {
    score += 10;
  }

  // Specialty match (max 15 points)
  if (
    mentee.preferences.has_transplant &&
    mentor.specialties.includes('lung_transplant')
  ) {
    score += 15;
  } else if (
    mentee.months_since_diagnosis <= 6 &&
    mentor.specialties.includes('newly_diagnosed')
  ) {
    score += 15;
  }

  // Gender preference (max 5 points)
  if (mentee.preferences.same_gender && mentor.gender === mentee.gender) {
    score += 5;
  } else if (!mentee.preferences.same_gender) {
    score += 3; // Doesn't care, so no penalty
  }

  return score; // Max score: 100
}

// Find top 3 matches for mentee
async function findMentorsForMentee(menteeId) {
  const mentee = await db.query('SELECT * FROM mentees WHERE id = $1', [
    menteeId
  ]);
  const availableMentors = await db.query(`
    SELECT * FROM mentors
    WHERE active = TRUE
    AND training_completed = TRUE
    AND current_mentees < max_mentees
  `);

  const scoredMentors = availableMentors.rows.map((mentor) => ({
    mentor,
    score: calculateMatchScore(mentor, mentee.rows[0])
  }));

  // Sort by score descending, return top 3
  scoredMentors.sort((a, b) => b.score - a.score);
  return scoredMentors.slice(0, 3);
}
```

```javascript
// Flag matches for review if issues detected
async function processCheckin(checkinData) {
  const { matchId, issuesReported, phq2Score } = checkinData;

  let flagForReview = false;

  // Flag if connection frequency is low
  if (checkinData.connection_frequency === '0') flagForReview = true;

  // Flag if helpfulness rating is low
  if (checkinData.helpfulness_rating <= 2) flagForReview = true;

  // Flag if PHQ-2 score indicates depression
  if (phq2Score >= 3) flagForReview = true;

  // Flag if specific issues reported
  if (
    issuesReported.includes('mismatch') ||
    issuesReported.includes('need_change')
  ) {
    flagForReview = true;
  }

  if (flagForReview) {
    await db.query(
      `
      UPDATE mentor_checkins SET flagged_for_review = TRUE WHERE id = $1
    `,
      [checkinData.id]
    );

    // Notify program coordinator
    await sendEmail(
      'coordinator@alpha1life.com',
      'Mentorship Needs Review',
      `
      Match ID ${matchId} has been flagged for review. Please follow up within 24 hours.
      Issues: ${issuesReported.join(', ')}
      PHQ-2 score: ${phq2Score}
    `
    );
  }
}
```
