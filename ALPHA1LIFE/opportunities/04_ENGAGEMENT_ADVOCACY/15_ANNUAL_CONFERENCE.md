## üìã Executive Summary

### The Problem

While virtual support offers connection, **nothing replaces in-person gatherings**. Alpha-1 patients need:

- **Face-to-face community**: Meeting others with Alpha-1 in person validates their experience, reduces isolation more powerfully than any video call
- **Comprehensive education**: Multi-day conferences with expert speakers (pulmonologists, geneticists, therapists), workshops, and Q&A sessions
- **Caregiver respite**: Conferences provide structured programming where caregivers can learn, connect, and recharge while patients do the same
- **Advocacy mobilization**: In-person gatherings galvanize political action (300 patients in one place = lobbying power)
- **Medical access**: Many patients travel to Alpha-1 Centers of Excellence for conferences, get consultations with top specialists while there
- **Hope**: Seeing severe patients living full lives inspires newly diagnosed patients ("If they can do it, I can too")

**Current gaps**:

- **No Alpha-1 specific national conference**: General lung disease conferences (ALA, COPD Foundation) lack Alpha-1 focus
- **Alpha-1 Foundation events**: Educational days (not multi-day conferences), limited programming, no virtual hybrid option
- **Geographic barriers**: Patients in rural areas can't afford travel, miss out on community
- **No year-round events**: One annual conference isn't enough‚Äîneed regional meetups, local support group gatherings
- **Poor accessibility**: No oxygen support coordination, mobility assistance, dietary accommodations

**Patient demand** (Alpha1Life survey, Oct 2025):

- **89% want in-person conference** (at least once in their lifetime)
- **67% would attend annually** if affordable (<$300 registration, hotel discounts)
- **72% want regional events** (1-day gatherings, 2-4 per year, within 200 miles)
- **81% want to bring family** (spouse, kids, parents‚Äîprogramming for all ages)
- **55% would attend virtually** if can't travel (hybrid model)

### The Solution

Build an **Annual Conference & Events Platform** that provides:

1. **Annual National Conference**: 3-day event (Friday-Sunday), 500-800 attendees, held in rotating Alpha-1 Center of Excellence cities (Nashville, Tampa, Portland, Cleveland)
2. **Regional Meetups**: 1-day events, 50-100 attendees, 10 regions/year (Southeast, Midwest, West Coast, etc.)
3. **Educational Workshops**: Expert-led sessions (treatment advances, clinical trials, mental health, nutrition, exercise, end-of-life planning)
4. **Family Programming**: Activities for kids, caregiver tracks, genetic counseling for at-risk relatives
5. **Networking Events**: Welcome reception, patient-only dinners, caregiver coffee chats, young adult happy hour
6. **Exhibit Hall**: Pharma sponsors (Grifols, CSL Behring), oxygen suppliers, travel companies, advocacy orgs
7. **Hybrid Model**: Livestream keynotes, virtual networking (for those who can't travel)
8. **Event Management Platform**: Registration, agenda, mobile app, attendee directory, session recordings

### Impact

**Year 1 Targets** (First Annual Conference):

- **600 attendees** (in-person) + 200 virtual
- **25 expert sessions** (keynotes, workshops, panels)
- **50 exhibitors** (sponsors, vendors)
- **$150K net revenue** (sponsorships cover costs + surplus for MEF programs)
- **4.8/5 satisfaction** ("Best event I've attended")

**Long-Term Impact**:

- **Stronger community**: Patients form lifelong friendships, support networks
- **Improved health literacy**: Attendees leave with actionable knowledge (treatment plans, clinical trial options)
- **Increased advocacy**: Conference attendees 3x more likely to contact legislators, participate in awareness campaigns
- **Donor engagement**: Conference attendees donate at 5x rate vs. non-attendees
- **Research recruitment**: Conferences are prime venues for enrolling patients in studies

### Registration System

```javascript
// Node.js - Handle conference registration
const stripe = require('stripe')(process.env.STRIPE_SECRET_KEY);

app.post('/api/conferences/:id/register', async (req, res) => {
  const { conferenceId } = req.params;
  const { userId, ticketType, dietaryRestrictions, oxygenNeeds } = req.body;

  // Check capacity
  const conference = await db.query('SELECT * FROM conferences WHERE id = $1', [
    conferenceId
  ]);
  const registrationCount = await db.query(
    'SELECT COUNT(*) FROM conference_registrations WHERE conference_id = $1',
    [conferenceId]
  );

  if (registrationCount.rows[0].count >= conference.rows[0].capacity) {
    return res.status(400).json({ error: 'Conference is full' });
  }

  // Calculate price
  const now = new Date();
  const earlyBirdDeadline = new Date(conference.rows[0].early_bird_deadline);
  const price =
    now <= earlyBirdDeadline
      ? conference.rows[0].early_bird_price
      : conference.rows[0].regular_price;

  // Create Stripe payment intent
  const paymentIntent = await stripe.paymentIntents.create({
    amount: price * 100, // Convert to cents
    currency: 'usd',
    metadata: {
      conferenceId,
      userId,
      ticketType
    }
  });

  // Save registration (pending payment)
  await db.query(
    `
    INSERT INTO conference_registrations
    (conference_id, user_id, registration_type, ticket_type, amount_paid, stripe_payment_intent_id, dietary_restrictions, oxygen_needs)
    VALUES ($1, $2, 'in_person', $3, $4, $5, $6, $7)
  `,
    [
      conferenceId,
      userId,
      ticketType,
      price,
      paymentIntent.id,
      dietaryRestrictions,
      oxygenNeeds
    ]
  );

  res.json({ clientSecret: paymentIntent.client_secret });
});
```

```javascript
app.post('/api/conferences/:id/scholarship', async (req, res) => {
```

- ‚úÖ Refund policy: Full refund up to 30 days before (reduces risk for attendees)

**Risk 2: Oxygen Supply Issues**

- Patients can't get oxygen at venue, medical emergencies
- Risk: Liability, unsafe environment

**Mitigation**:

- ‚úÖ Pre-event oxygen survey: Collect oxygen needs (flow rate, portable vs. continuous) 60 days out
- ‚úÖ Partner with local supplier: Lincare or Inogen provides backup tanks on-site
- ‚úÖ Oxygen-friendly venue: Hotel provides electrical outlets every 20 feet (for portable concentrators)
- ‚úÖ Emergency protocol: On-site nurse, AED, 911 plan

**Risk 3: Medical Emergency at Conference**

- Patient collapses, exacerbation, respiratory distress
- Risk: Liability, trauma for attendees

- ‚úÖ On-site medical staff: Registered nurse + EMT volunteer (both Alpha-1 patients, know the disease)

- ‚úÖ Vaccination requirement: Proof of COVID-19, flu, pneumonia vaccines (or medical exemption)
